文档

基于layui.tree的树形下拉选择组件，支持单选和多选。

示例如下：


使用方法： 
layui.use(['customSelect'], function() {
    const select = layui.customSelect
    select.render({
        el: 'searchBox',
        data: list,
        checkbox: true,
        checked: function (obj) {
            console.log(obj)
           //obj.value 已选数据的title
           //obj.data 已选数据
        }
    })
})
