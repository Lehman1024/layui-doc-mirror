iconPicker 图标选择器 iconPicker

无求**丶

基于 Layui 的图标选择器，支持查询、点击回调等功能，提供手动选择图标事件。注：图标来自于 Layui 2.4.1 https://wujiawei0926.gitee.io/iconpicker/demo.html

文档

具体信息请前往码云查看
https://wujiawei0926.gitee.io/iconpicker/demo.html
使用示例： 
<input type="text" id="iconPicker" lay-filter="iconPicker" class="layui-input">
    layui.use(['iconPicker'], function () {
        var iconPicker = layui.iconPicker;
            
            iconPicker.render({
                // 选择器，推荐使用input
                elem: '#iconPicker',
                // 数据类型：fontClass/unicode，推荐使用fontClass
                type: 'fontClass',
                // 是否开启搜索：true/false
                search: true,
                // 点击回调
                click: function (data) {
                    console.log(data);
                }
            });

            /**
             * 选中图标 （常用于更新时默认选中图标）
             * @param filter lay-filter
             * @param iconName 图标名称，自动识别fontClass/unicode
             */
            iconPicker.checkIcon('iconPicker', 'layui-icon-star-fill');

    });
注意事项：
1. 推荐使用fontClass
