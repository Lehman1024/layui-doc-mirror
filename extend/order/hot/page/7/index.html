    # NumberInput 数字输入框 扩展了 Layui 默认 input 的功能, 给 input 增加输入数值的功能, 换个简单的方式来说吧! 本扩展可以更方便的让你在管理后台或类似场景中, 更加快速便捷的输入如: 排序号,价格,数量...等等值! ## 如何使用 1. 前往 [Releases](https://gitee.com/layui-exts/number-input/releases) 页面下载最新版! 2. 解压到你项目中 Layui 的扩展目录, 并确认配置无误! 3. 页面引入本扩展必要的样式文件, 然后就可以开始使用了!
