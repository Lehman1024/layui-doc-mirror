文档

str 要转为拼音的汉字
separa 分隔符号，默认为空格
percase 1=每个汉字首个英文大写，2=全部大写，默认为全小写
layui.strpy(str,separa,percase);
例: 
layui.config({
  base: './style/' //配置 layui 第三方扩展组件存放的基础目录
}).extend({
    strpy: "strpy",
}).use(["strpy"], function () {
    alert(layui.strpy('汉字转拼音'));
    alert(layui.strpy('汉字转拼音','|',1));
    alert(layui.strpy('汉字转拼音','@',2))
    alert(layui.strpy('汉字转拼音',' ',1));
    alert(layui.strpy('汉字转拼音',' ',2));;
});
