文档

效果图


1. 引入模块, 
    // 加载模块
    layui.config({
        base: 'module/' //假设这是你存放拓展模块的根目录
    }).extend({ //设定模块别名
        sysdata: 'sysdata'
    });
2. 组件里的而方法清单, 详细的参数说明,源码注释上有 
createKey : 生成一个数据保存的key
saveData : 保存数据的方法,数据格式 {param:{},result:{},obj:{}}
saveDataParam : 保存数据的方法, 直接保存到 param
saveDataResult : 保存数据的方法, 直接保存到 result
saveDataObj : 保存数据的方法, 直接保存到 obj
getData : 获取全部的数据
getDataParam : 获取数据里面  param 的值
getDataResult : 获取数据里面 result 的值
getDataObj : 获取数据里面 obj 的值
cleanData : 清除数据
作者寄语 : 一个很简单的组件,上面的demo只是它的一个简单用法,如果用的好, 它还能产生更大的作用哦
